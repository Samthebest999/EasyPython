# simplpython Docs
## Installation:

1. Run `pip install simplpython`
1. In your python terminal or script type `from simplpython import * as sp`
1. Test by running `sp.display("Hi, Does this work?")` if the output is `Hi, Does this work?` It **WORKED!**
